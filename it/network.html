Network

ネットワークは、端末同士がルールに従ってデータを届け合う仕組み。Web、クラウド、セキュリティの土台になる。

まず見る流れ

ブラウザでURLを開くと、名前解決、接続、暗号化、リクエスト、レスポンスという順番で通信が進む。最初は細かいプロトコル名よりも、どこで何が決まっているかを押さえる。

Example

example.com を開くと、DNSでIPアドレスを調べ、TCPで通信路を作り、HTTPSで暗号化してからHTTPリクエストを送る。

基本キーワード

  • IP address: ネットワーク上の住所。
  • DNS: ドメイン名をIPアドレスに変換する仕組み。
  • TCP: データを順番通り、欠けにくく届ける通信方式。
  • HTTP: WebブラウザとWebサーバーが情報をやり取りするルール。
  • Router: 別のネットワークへデータを中継する機器。

次に深掘りしたいこと

OSI参照モデル、TCP/IP、サブネット、ポート番号、HTTPステータスコード、DNSレコードを順番に足していく。
