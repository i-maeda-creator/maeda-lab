Quick Quiz 1

「ログイン済みだが売上画面は見られない」に近い問題はどれでしょうか。

本人確認は済んでいますが、権限が足りません。認可の問題です。

Applied Quiz 2

API Gateway経由だけ401になります。直接アプリへ送ると200です。まず疑うものはどれでしょうか。

Gatewayを通した時だけなら、認証設定やヘッダーの扱いが怪しいです。

このページの落ち

ログインは入口、権限は部屋の鍵です。入れたのに見られない、という障害はここで起きます。

Auth / Session 午後記述

短答記述

SSO後に特定利用者だけ無限リダイレクトします。認証連携で確認するパラメータを答えてください。

事例ベースの午後問題

社内IdPとSaaSのSAML連携で、社内は成功、社外はログイン後に戻れません。Reply URL、Cookie、時刻同期、プロキシを含めて説明してください。

模範解答・採点観点

  • Reply URL、Issuer、Audience、時刻同期、Cookie属性、プロキシヘッダを挙げる。
  • 社内外でLocationとCookieを比較し、IdP/SP双方のログを見る。
  • ログイン成功と認可成功、セッション確立を分けて追う。
