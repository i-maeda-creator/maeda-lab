Quick Quiz 1

注文一覧を取得する操作に最も近いHTTPメソッドはどれでしょうか。

取得はGETです。作成はPOST、更新はPUT/PATCH、削除はDELETEが代表です。

Applied Quiz 2

ログイン後の管理画面だけ403になります。まず見るべきものはどれでしょうか。

403は権限やアクセス制御の問題です。ログイン済みか、許可されているかを見ます。

このページの落ち

HTTPはWebの操作そのものです。画面の裏では、注文票が決まった形式で行き来しています。

Network Specialist Written

HTTP 午後記述

短答記述

ブラウザでは成功するがAPIクライアントでは415です。HTTPで確認すべき要素を答えてください。

事例ベースの午後問題

注文APIだけタイムアウトします。GETは200、POSTは本文サイズが大きいと失敗します。LB、WAF、アプリで確認する観点を説明してください。

模範解答・採点観点

  • メソッド、Content-Type、Accept、本文形式、認証ヘッダ、ステータスコードを挙げる。
  • ボディ制限、WAFルール、ヘルスチェックとの差、処理時間を見る。
  • 200だけで正常判断せず業務パスを再現する。
